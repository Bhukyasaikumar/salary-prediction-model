# Salary Prediction Model

This project predicts salary based on features like experience, education, job title, and location using Linear Regression.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Model Performance
- R² Score: 0.87
- Mean Squared Error: 105996277

## How to Run
1. Install dependencies:
   pip install pandas numpy scikit-learn

2. Run:
   python salary_prediction.py

## Output
- salary_model.pkl
- predictions.csv
